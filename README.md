# Carbon Field: uniqid

Provides an unique ID field for Carbon Fields

Adds a `uniqid` field type to Carbon Fields. Install using Composer:

```cli
composer require iamntz/carbon-uniqid
```

Usage:

```php
Field::make('uniqid', ....)
```
